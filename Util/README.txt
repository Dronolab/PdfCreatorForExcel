Pour installer le AddIn :
0) Attention, vous devez avoir Excel 2010!
1) copier le dossier Application Files, le fichier PdfCreatorForExcel.vsto et le fichier setup.exe sur le bureau
2) double click sur setup.exe
3) un fois installer, Ouvrez Excel 2010 et vous devriez avoir un Ribbon (menu en haut) PDF
4) Quand vous avez fini la feuille, vous pouvez allez dans le menu PDF et peser créer le PDF
5) Lors de la premiere utilisation, vous devez entrer un dossier ou les PDF seront stocké et un liens vers le PDF template (Template.pdf)
